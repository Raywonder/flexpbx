    • üîÑ Updating Documentation

    Converting Markdown to HTML

    A PHP script is available to convert .md files to HTML:

    php /home/flexpbxuser/public_html/docs/markdown-to-html.php

    This will:

  • Find all .md files in /docs/

  • Convert them to HTML with proper styling

  • Create downloadable links

  • Update the documentation portal

    • Adding New Documentation


  • Create your .md file in /home/flexpbxuser/public_html/docs/

  • Run the conversion script (above)

  • Add entry to index.html if needed

  • Set permissions: chmod 644 filename.md filename.html
