API Endpoints

GET /api/info
Get API information and available endpoints

Authentication: None required

GET /api/status
Get real-time system status and performance metrics

Authentication: None required

{ "success": true, "data": { "uptime": "up 2 days, 14 hours, 23 minutes", "load_average": 0.45, "memory_usage": 128.5, "cpu_usage": 45.0, "extensions": 12, "active_calls": 3, "api_requests": 1247, "timestamp": "2024-10-11T22:30:00Z" } }
GET /api/extensions
List all PBX extensions with their current status

Authentication: Required

X-API-Key string
Your API authentication key

POST /api/extensions
Create a new PBX extension

Request Body Parameters:

name string
Display name for the extension
email string
Contact email (optional)
{ "success": true, "message": "Extension created successfully", "data": { "id": 1004, "name": "New Extension", "status": "created" } }

Retrieve system logs and events

Authentication: Required

limit integer

Usage Examples

cURL Examples

# Get API information (no auth required) curl https://flexpbx.devine-creations.com/api/info # Get system status (no auth required) curl https://flexpbx.devine-creations.com/api/status # List extensions (authentication required) curl -H "X-API-Key: flexpbx_api_2024" \ https://flexpbx.devine-creations.com/api/extensions # Create new extension curl -X POST \ -H "Content-Type: application/json" \ -H "X-API-Key: flexpbx_api_2024" \ -d '{"name":"Sales Team","email":"sales@company.com"}' \ https://flexpbx.devine-creations.com/api/extensions # Get recent logs curl -H "X-API-Key: flexpbx_api_2024" \ "https://flexpbx.devine-creations.com/api/logs?level=INFO&limit=5"

JavaScript/Fetch Examples

// Get system status fetch('https://flexpbx.devine-creations.com/api/status') .then(response => response.json()) .then(data => console.log(data)); // List extensions with authentication fetch('https://flexpbx.devine-creations.com/api/extensions', { headers: { 'X-API-Key': 'flexpbx_api_2024' } }) .then(response => response.json()) .then(data => console.log(data)); // Create backup fetch('https://flexpbx.devine-creations.com/api/backup', { method: 'POST', headers: { 'X-API-Key': 'flexpbx_api_2024', 'Content-Type': 'application/json' } }) .then(response => response.json()) .then(data => console.log('Backup created:', data));
