## 🏗️ TECHNICAL ARCHITECTURE

### Service-Oriented Design
```
FlexPBX v1.0 Architecture:
├── UnifiedServiceManager (Orchestration)
├── RemoteAccessibilityService (Screen readers + Audio)
├── RichMessagingService (Communication + Encryption)
├── CopyPartyService (File sharing + Security)
├── DNSManagerService (DNS management + Sync)
├── SoftwareUpdateService (Remote updates)
└── SoundManager (Audio feedback)
```

### Real-Time Communication
- **WebSocket Servers**: Multiple endpoints (41237, 41238)
- **Message Queuing**: Reliable offline delivery
- **Health Monitoring**: Auto-restart and recovery
- **Event Architecture**: Reactive component updates

## 📦 DEPLOYMENT OPTIONS

### 🔌 **Portable Edition** (RECOMMENDED)
Perfect for detachable drives, no installation needed:

1. **Download**: `FlexPBX-v1.0-Portable.zip`
2. **Extract**: To your USB/external drive
3. **Run**: Platform-specific launcher
   - Windows: `FlexPBX-Portable.bat`
   - macOS: `FlexPBX-Portable.command`
   - Linux: `FlexPBX-Portable.sh`
4. **Enjoy**: Instantly works on any computer!

### 🖥️ **Standard Installation**
Traditional installation for permanent setups:

- **macOS**: `FlexPBX-v1.0-mac-universal.dmg`
- **Windows x64**: `FlexPBX-v1.0-win-x64.exe`
- **Linux**: `FlexPBX-v1.0-linux-x64.AppImage`
