Quick Installation

Method 1: Automatic (Recommended)

  1. Download the configuration automatically
  2. Append to your musiconhold.conf
  3. Reload Asterisk MOH module
# Download and append MOH configuration curl -s "https://flexpbx.devinecreations.net/api/moh-provider.php?action=config&format=asterisk" >> /etc/asterisk/musiconhold.conf # Reload MOH module asterisk -rx "moh reload" # Verify streams are loaded asterisk -rx "moh show classes"

Method 2: Manual Installation

  1. Edit /etc/asterisk/musiconhold.conf
  2. Add desired stream configurations
  3. Save and reload
# Example: Add Raywonder Radio (Audio Described) [raywonder-radio] mode=custom application=/usr/bin/ffmpeg -i https://stream.raywonderis.me/jellyfin-radio -f s16le -ar 8000 -ac 1 - format=slin ; Audio Described TV, Movies, and Music
